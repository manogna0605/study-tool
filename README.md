# StudySmart

StudySmart is a beginner-friendly study tool built for students to save time and reduce exam stress.

## Problem
Students waste a lot of time searching for reference materials and struggle to plan their exam preparation properly.

## Solution
StudySmart provides:
- A Reference Finder to quickly access trusted study materials
- An Automatic Exam Timetable Generator based on exam dates and difficulty

## Features
- Search reference papers and notes easily
- Generate a simple study timetable
- Clean and easy-to-use interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open `index.html` in any web browser
